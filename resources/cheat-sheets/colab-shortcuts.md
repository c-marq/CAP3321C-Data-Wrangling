# Google Colab Keyboard Shortcuts

Speed up your workflow with these shortcuts.

---

## Running Cells

| Action | Shortcut |
|--------|----------|
| Run cell and move to next | `Shift + Enter` |
| Run cell and stay | `Ctrl + Enter` |
| Run all cells | `Ctrl + F9` |
| Interrupt execution | `Ctrl + M` then `I` |

---

## Cell Navigation

| Action | Shortcut |
|--------|----------|
| Move to cell above | `Ctrl + M` then `P` |
| Move to cell below | `Ctrl + M` then `N` |
| Go to beginning | `Ctrl + Home` |
| Go to end | `Ctrl + End` |

---

## Cell Operations

| Action | Shortcut |
|--------|----------|
| Add code cell below | `Ctrl + M` then `B` |
| Add code cell above | `Ctrl + M` then `A` |
| Convert to text cell | `Ctrl + M` then `M` |
| Convert to code cell | `Ctrl + M` then `Y` |
| Delete cell | `Ctrl + M` then `D` |
| Undo delete | `Ctrl + M` then `Z` |
| Copy cell | `Ctrl + M` then `C` |
| Paste cell | `Ctrl + M` then `V` |

---

## Editing

| Action | Shortcut |
|--------|----------|
| Undo | `Ctrl + Z` |
| Redo | `Ctrl + Y` |
| Find | `Ctrl + F` |
| Find and replace | `Ctrl + H` |
| Comment/uncomment line | `Ctrl + /` |
| Indent | `Tab` |
| Unindent | `Shift + Tab` |
| Select all | `Ctrl + A` |

---

## Code Completion

| Action | Shortcut |
|--------|----------|
| Autocomplete | `Tab` |
| Show tooltip/docs | `Shift + Tab` (after function name) |

---

## View

| Action | Shortcut |
|--------|----------|
| Show keyboard shortcuts | `Ctrl + M` then `H` |
| Toggle line numbers | `Ctrl + M` then `L` |
| Focus on editor | `Escape` |

---

## Tips

1. **Two-key shortcuts**: Press `Ctrl + M`, release, then press the second key
2. **Command palette**: `Ctrl + Shift + P` to search for any command
3. **Tab completion**: Type partial code and press `Tab` to autocomplete
