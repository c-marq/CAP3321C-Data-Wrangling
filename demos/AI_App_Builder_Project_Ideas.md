## Data Wrangling Applications

### 1. Data Cleaning Assistant
- Upload CSV with messy data
- AI identifies issues (missing values, duplicates, inconsistent formats)
- Suggests cleaning strategies from class (drop, fill, standardize)
- Shows before/after preview
- **Context:** Miami small business with messy customer spreadsheet

### 2. Dataset Quality Checker
- Upload or paste sample data
- AI evaluates completeness, consistency, accuracy
- Assigns "readiness score"
- Provides prioritized cleanup to-do list
- **Context:** Quality control before analysis projects

### 3. CSV Column Mapper
- Upload two datasets that need merging
- AI suggests which columns should match
- Identifies data type mismatches
- Recommends merge strategy (inner, outer, left, right)
- **Context:** Combining data from different South Florida business systems

### 4. Data Type Detective
- Paste a column of data
- AI identifies the actual data type (even if stored wrong)
- Suggests proper pandas dtype
- Shows conversion code
- **Context:** Fixing imported data with wrong types
